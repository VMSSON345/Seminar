# RecipeRAG GitHub Pages Slides

## Cách deploy lên GitHub Pages

1. Tạo repository mới, ví dụ: `reciperag-slides`.
2. Upload toàn bộ file trong thư mục này lên repository: `index.html`, `styles.css`, `script.js`, thư mục `assets/`.
3. Vào **Settings → Pages**.
4. Source: chọn **Deploy from a branch**.
5. Branch: chọn `main`, folder chọn `/root`.
6. Mở link GitHub Pages được tạo.

## Điều khiển slide

- Mũi tên phải / Space: slide tiếp theo.
- Mũi tên trái: quay lại.
- Home / End: slide đầu / cuối.
- In hoặc xuất PDF: dùng chức năng Print của trình duyệt.
